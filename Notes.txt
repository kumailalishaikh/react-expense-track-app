React Router:
    - React router gives components to create client side routing.
    - Without describing path property inside <Route /> the component will render on every html page.
    - Query string: '?query=querystring'

Redux:
    - Action generators are functions that return action objects.
    - Reducers are pure functions (outputs are determined by inputs).
    - In reducer never change state or action directly.
    - combineReducers() methods allows to combine multiple reducers.

React with Redux:
    - Higher Order components [HOC] is regular component with renders other component.
    - Advantages of HOC are: -
        - Reuse code 
        - Render Hijacking
        - prop manipulation
        - Abstract state
